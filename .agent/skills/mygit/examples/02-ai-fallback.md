# mygit 示例 - AI 失败托底

## 场景

AI API 返回错误（如 429 限流），脚本自动降级使用托底逻辑生成提交信息。

## 输入

```bash
./scripts/mygit.sh
```

## AI 请求失败

```text
API 请求失败: 429
```

## 托底逻辑

脚本自动根据变更统计生成提交信息：

```typescript
const dateStr = new Date().toISOString().split('T')[0];  // "2026-03-19"
commitMessage = `chore: 自动同步代码变更 (${dateStr})\n\n变更摘要：\n- ${changeParts.join('\n- ')}\n\n由于 AI 生成失败，此信息由系统自动生成。`;
```
