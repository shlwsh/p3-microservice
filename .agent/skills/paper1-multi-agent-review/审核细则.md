**自动化验证命令**（Phase 1 预检员 + 学术编辑必跑）：

```bash
# F1: 表格 caption 必须在 tabular 之前（源码检查）
echo "=== F1: 表题位置 ==="
for f in latex/sections/zh/*.tex; do
  awk '/\\begin\{table/{intab=1;cap=0} intab&&/\\begin\{tabular/{if(!cap) print FILENAME": ❌ caption在tabular之后"} intab&&/\\caption/{cap=1} /\\end\{table/{intab=0;cap=0}' "$f"
done

# F3: 图片 PDF 无内嵌标题
echo "=== F3: 图片内嵌标题 ==="
for f in figures/fig*.pdf; do
  cnt=$(pdftotext "$f" - 2>/dev/null | grep -c '图[0-9]')
  [ "$cnt" -gt 0 ] && echo "$(basename $f): ❌ 内嵌标题" || echo "$(basename $f): ✅"
done

# F6: 绘图脚本无残留标题调用
echo "=== F6: 绘图脚本标题残留 ==="
grep -n 'set_title\|suptitle' figures/plot_*.py && echo "❌ 发现残留" || echo "✅ 无残留"
```

**Major Issue 触发**：F1–F4 任一不通过即报 `FORMAT-F*`（P0）。

**防范规则**：

1. 新增表格：`\caption` + `\label` 紧跟 `\centering` 之后，`\begin{tabular}` 之前
2. 新增图片：`\caption` + `\label` 紧跟 `\includegraphics` 之后，`\end{figure}` 之前
3. 绘图脚本不使用 `set_title()`/`suptitle()` 嵌入标题；标题由 LaTeX `\caption` 管理
4. 每次生成/更新图片后用 `pdftotext` 验证无嵌入标题
