## J1–J7 — 《软件学报》投稿专检（`TARGET_JOURNAL=JOS` 时必跑）

> 依据官网投稿指南与「敬告作者」（2026）。细则见 [软件学报投稿要点.md](软件学报投稿要点.md)。

| ID | 严重性 | 描述 | 验收 | 主责 |
|----|--------|------|------|------|
| J1 | P1 | 稿内暗示已发表/一稿多投风险 | 无「已发表于」「同时投」等表述 | 📐 |
| J2 | **P0** | **创新性不足**：未与国内外同类工作充分比较 | 相关工作有分项对比；非仅罗列 | 🔬 |
| J3 | **P0** | **国内文献不足**或未在正文对比 | C4c + 正文国内 work 对比段 | 🔬 |
| J4 | P1 | 方法缺实验或形式化支撑（「优越」无据） | 表1/算法节/phase3 可对应 | 🔍 |
| J5 | P1 | 像产品报告/科普，软件技术含量低 | 无营销语；核心为方法与系统 | 🔬 ✍️ |
| J6 | P1 | 未用 JOS 模板或图表无题注/引用 | `main-jos.tex` + `\caption` + `\ref` | ✍️ |
| J7 | P2 | 中英摘要/关键词不完整 | `main-jos.tex` 中英摘要齐全 | ✍️ |

**JOS 快速命令**：

```bash
test -f latex/main-jos.tex
grep -Ein "本产品|国内领先|唯一|填补空白" latex/sections/zh/ docs/v4-论文稿件.md || true
python3 -c "
import re, pathlib
bib=pathlib.Path('latex/references.bib').read_text(encoding='utf-8',errors='ignore')
es=re.split(r'@\w+\{',bib)[1:]
zh=sum(1 for e in es if re.search(r'[\u4e00-\u9fff]',e) or '软件学报' in e or '计算机学报' in e)
print('bib entries',len(es),'zh-ish',zh,'ratio',round(zh/max(len(es),1),2))
"
```
