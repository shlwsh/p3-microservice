# 评审技能顶刊规范更新报告

生成时间：2026-06-09 14:32:55

## 一、更新目标

本次更新将“计算机科学顶级期刊论文规范指南与评审要求”纳入 `paper1-multi-agent-review` 技能，使其不仅能完成 JOS 投稿前检查和基准论文 A/B 对照，还能系统审查顶级期刊常见的五类要求：稿件评审、整体结构、行文表达、图表公式、参考文献著录。

## 二、主要调整

1. 新增 `顶级期刊论文规范指南.md`，将用户提供的五维规范整理为可执行规则，并给出快速裁决口径。
2. 更新 `SKILL.md`，把顶刊规范列入启动必读文档和 Definition of Done，新增 T1–T5 检查项。
3. 更新 `审核细则.md`，在预检、方法论、领域、统计、编辑、伦理和 Phase 3 交互质询中加入顶刊检查规则。
4. 更新 `已知问题清单.md`，新增 T1–T5 专检问题、严重性、验收标准和主责角色。
5. 更新 `reference.md` 和 `README.md`，同步输出报告目录、Issue 前缀、触发语和文档索引。

## 三、能力提升

- 引言审查从“结构是否完整”提升为“痛点、贡献、RQ/SQ、证据链是否闭环”。
- 实验审查新增 RQ/SQ 到指标、图表和结论的映射要求。
- 写作审查新增摘要四要素、客观书面语、实验原因分析和贡献贯穿一致性。
- 图表公式审查新增公式编号、符号解释、表题图题位置、三线表、最优结果加粗等形式门禁。
- 参考文献审查新增中文文献英译或 `in Chinese`、文献类型标识、DOI/URL、归档真实性等要求。

## 四、后续使用方式

执行论文多角色审核时，预检员应同时检查 C1–C6、J1–J7（若投 JOS）和 T1–T5。若 T1/T4/T5 任一为 P0，Phase 3 应加入 `CL-TOP-RQ` 或 `CL-TOP-FMT` 议题，并在 `05-全面评审报告.md` 中给出顶级期刊五维规范检查结论。
