## 二、修正方案与操作细节

### F7：表格标题位置（P0）

**问题**：中文学术期刊（含《软件学报》）要求**表题在表上方**（table caption above），但 5 个表格的 `\caption` 均位于 `\end{tabular}` 之后、`\end{table}` 之前（即在表格下方）。

**修正方法**：将 `\caption{...}` 和 `\label{...}` 移至 `\centering` 之后、`\begin{tabular}`（或 `\resizebox`）之前。

**涉及文件与修正**：

| 文件 | 表格 | 修正前位置 | 修正后位置 |
|------|------|------------|------------|
| `02_related.tex` | 表1（方案对比） | `\end{tabular}` 之后 | `\centering` 之后 |
| `03_system.tex` | 表2（三层输入/输出） | `\end{tabular}` 之后 | `\centering` 之后 |
| `05_implementation.tex` | 表3（技术选型） | `\end{tabular}` 之后 | `\centering` 之后 |
| `06_experiments.tex` | 表4（RQ映射） | `\end{tabular}` 之后 | `\centering` 之后 |
| `06_experiments.tex` | 表5（资源对比） | `\end{tabular}` 之后 | `\centering` 之后 |

**LaTeX 规范结构**：

```latex
% ✅ 正确：caption 在上
\begin{table}[htbp]
  \centering
  \caption{表格标题}          % ← 在上方
  \label{tab:xxx}
  \begin{tabular}{...}
    ...
  \end{tabular}
\end{table}

% ❌ 错误：caption 在下
\begin{table}[htbp]
  \centering
  \begin{tabular}{...}
    ...
  \end{tabular}
  \caption{表格标题}          % ← 在下方（图可以，表不行）
  \label{tab:xxx}
\end{table}
```

### F8：图片双标题（P0）

**问题**：matplotlib 绘图脚本中使用 `ax.set_title()` / `fig.suptitle()` 为每张图嵌入了"图X 标题内容"，导致 PDF 图片本身自带标题。加上 LaTeX `\caption{...}`，最终 PDF 输出出现双标题。

**修正方法**：

1. **修改绘图脚本**：删除所有 `set_title()` / `suptitle()` / `ax.text(... title ...)` 调用
2. **重新生成图片 PDF**：运行修改后的脚本
3. **保持 LaTeX `\caption`**：由 LaTeX 统一管理编号和标题

**涉及文件**：

| 文件 | 受影响的图 | 删除的调用 |
|------|-----------|------------|
| `figures/plot_all_phase1.py` | fig1–fig6 | `set_title`×4, `suptitle`×1, `ax.text`×1 |
| `figures/plot_phase4.py` | fig7–fig8 | `suptitle`×1, `set_title`×1 |

**根因分析**：绘图脚本最初为独立预览而设计，包含了完整的中文标题（如"图1 系统总体架构"）。嵌入 LaTeX 论文后，应由 LaTeX `\caption` 管理标题，绘图脚本只输出纯图表内容。

**检查命令**：

```bash
# 检查图片 PDF 是否仍有嵌入标题（首行应为图表数据，非"图X"）
for f in figures/fig*.pdf; do
  echo "$(basename $f): $(pdftotext "$f" - 2>/dev/null | head -1)"
done

# 检查绘图脚本是否残留 set_title/suptitle
grep -n 'set_title\|suptitle' figures/plot_*.py
```

## 四、经验教训与防范建议

1. **绘图脚本与 LaTeX 分离标题**：matplotlib 脚本不应包含 `set_title`/`suptitle`，标题统一由 LaTeX `\caption` 管理。建议在绘图脚本中加注释提醒。

2. **表/图标题位置规则**：中文学术期刊统一规范——**表题在上、图题在下**。每次新增 table/figure 环境时须检查。

3. **编译后 PDF 验证**：每次编译后应用 `pdftotext` 抽检关键格式点（标题位置、编号顺序、双标题），避免仅查看 `.tex` 源码而忽视实际输出。

4. **模板对照**：格式调整前先通过 `antiword` 或 `python-docx` 解析官方 `.docx` 模板，提取字号、行距等具体参数，不要凭感觉调整。

5. **纳入评审技能**：已将上述格式检查点纳入 `paper1-multi-agent-review` 技能的审核细则（F-FORMAT 系列），确保后续评审自动覆盖。
