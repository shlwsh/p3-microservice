# DOCX 页码、图表与表格边框修正报告

生成时间：2026-06-13 09:19:50

## 1. 本轮修正的问题

| 问题 | 根因 | 修正 |
|---|---|---|
| 页码没有带入 | header XML 只有纯文本，没有 Word `PAGE` 字段 | 在奇偶页页眉写入 `PAGE` 字段；首页保持 `titlePg` 无页眉 |
| 首页期刊信息行格式不正确 | 用空格模拟左右栏，Word 中不稳定 | 改为右对齐制表位，前三行均以 `w:tab` 对齐右侧邮箱/网址/电话 |
| 末尾作者简介乱码 | LaTeX `list/description` 环境参数被当成正文条目 | 附中文参考文献和作者简介都从第一个 `\item` 开始提取，跳过环境参数 |
| 参考文献左右页边距不正确 | `JOSReference` 只有 hanging 缩进，缺少 left 基准 | 改为 `left=420, hanging=420`，首行从正文左边界开始，续行悬挂缩进 |
| 图 5、图 6 被正文覆盖 | 图片段落继承 `Normal` 固定 13pt 行距 | 新增 `JOSImage` 段落样式，不设置固定行距；所有图片段落使用该样式 |
| 表格中间竖线缺失 | 表格 `insideV` 写成 `nil` | 改为 `insideV=single`；左右外边框仍为 `nil` 保持开口 |
