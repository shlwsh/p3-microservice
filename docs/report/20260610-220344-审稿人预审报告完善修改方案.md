## 三、具体修改方案

### P0：立即修改项（投稿底线）

#### P0-1：确认并修复引言笔误

- **文件**：`latex/sections/zh/01_intro.tex`
- **操作**：全文 grep 确认无「hi微服务」或类似笔误残留；若发现其他排版错误一并修复
- **说明**：审稿人指出「hi微服务架构」笔误。经检查当前稿件首段已无「hi」字样（疑似先前版本已修复），仍需再次全文确认

#### P0-2：发射频率混淆 → 因果归因强化

> **共识最高优先级**。三位审稿人一致认为：定向 2s vs 全量 400ms 使 98.4% 降幅混合了「源速率」与「过滤策略」，核心因果链未隔离。

**修改策略**：在无法立即补跑同频率实验的前提下，采用 **理论分解 + 敏感性定界 + 叙事克制** 三管齐下。

| 文件 | 修改内容 |
|------|---------|
| `06_experiments.tex` §6.3 | 「发射频率敏感性」段落扩展为独立小节，增加定量分解模型 |
| | - 频率比 $r = 2000/400 = 5$；同频率下全量预期 $\approx 878$ 条 |
| | - 定向模式 72 条 → 同频率降幅 $\approx 91.8\%$ |
| | - **结论**：策略过滤贡献至少 91%，频率差异贡献约 6.6 个百分点 |
| | - 增加敏感性边界声明 |
| `main-jos.tex` 摘要 | 中英文摘要中 98.4% 后补充联合效果与策略过滤保守估计说明 |
| `01_intro.tex` 贡献 (3) | 98.4% 引用加注联合效果说明 |
| `07_conclusion.tex` | 降低结论力度，改为引用「百条量级」+「保守估计同频率降幅不低于 91%」 |
