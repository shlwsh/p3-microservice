## 2. 主对比表（三期）

数据源：`experiments/results/phase3/phase3_latest.json` → `comparison`。

| 指标 | 定向采集 | 全量采集 | 降低比例 |
|------|----------|----------|----------|
| Loki 入库日志 | 72±0 条 | 4388±1 条 | **98.4%** |
| Agent CPU | 0.05±0.01% | 0.08±0.02% | 37.5% |
| Agent 内存 | 95.6±4.2 MB | 97.5±1.7 MB | 2.0% |
| 带宽估计 | 0.20 KB/s | 12.19 KB/s | 98.4% |
| 关注清单 | 16 个 URL 模式 | — | — |

解读：

1. 主 claim 是 Loki 入库量下降，不是 CPU 大幅下降。
2. 全量标准差为 ±1 条，不能写成 ±1000。
3. 全量模式应用日志发射间隔为 400 ms，定向模式为 2 s；这是策略过滤与源配置差异的联合效果，论文必须主动披露。
