# 04 · 系统架构与代码地图

## 1. 仓库总览

```
p3-microservice/
├── agent/          # Sidecar 采集器（Go）
├── center/         # 日志中心（Go）
├── proto/          # gRPC / Protobuf 定义
├── deploy/         # Docker Compose + OpenResty + WSL 脚本
├── experiments/    # 实验脚本、配置、L0 结果
├── figures/        # 论文图表 + plot 脚本
├── latex/          # LaTeX 中英源稿
├── docs/           # L1 文档 + study（本目录）
├── scripts/        # 一键科研流水线 p3_run_phase*.sh
└── reviews/        # 多智能体审核报告
```

## 2. Agent 模块地图

入口：`agent/cmd/agent/main.go`

| 包 | 文件 | 职责 |
|----|------|------|
| `config` | `config.go` | YAML 配置加载 |
| `collector` | `collector.go` | 应用日志采集、发射间隔控制 |
| `matcher` | `rule_matcher.go` | 关注清单 URL/级别匹配 |
| `cache` | `ring_buffer.go`, `cache_block.go` | 固定缓存块 |
| `retry` | `backoff.go` | 指数退避 |
| `uploader` | `grpc_uploader.go` | gRPC 上传 + BoltDB 兜底 |
| `monitor` | `pressure.go` | CPU/内存压力感知 |
| `metrics` | `prometheus.go` | Prometheus 指标 |

配置样例：

- 网关 Agent：`deploy/docker/gateway-agent-config.yaml`
- 微服务 Agent：`deploy/docker/service-agent-config.yaml`

## 3. Center 模块地图

入口：`center/cmd/center/main.go`

| 包 | 文件 | 职责 |
|----|------|------|
| `strategy` | `high_value_filter.go` | 高价值流量过滤 |
| `strategy` | `url_cluster.go` | URL 聚类泛化 |
| `strategy` | `attention_list.go` | 关注清单生成 |
| `strategy` | `triple_transform.go` | **三次转换引擎** |
| `redisstore` | `gateway_log_store.go` | Redis 网关日志读写 |
| `dispatch` | `rule_dispatch.go` | 规则广播至 Agent |
| `receiver` | `log_receiver.go` | 日志接收、去重、二次过滤 |
| `storage` | `loki.go` | Loki Push API |
| `grpcserver` | `server.go` | gRPC 服务（上传 + 清单拉取） |

HTTP API（调试/实验）：

| 路径 | 说明 |
|------|------|
| `/api/v1/health` | 健康检查 |
| `/api/v1/attention-list` | 当前关注清单 |
| `/api/v1/agents` | 已注册 Agent 列表 |
| `/api/v1/stats` | 二期统计接口 |

## 4. gRPC 协议

## 5. 部署拓扑（Docker Compose）

```
                    ┌─────────────┐
                    │  OpenResty  │ :8088
                    │  (网关)      │
                    └──────┬──────┘
                           │
         ┌─────────────────┼─────────────────┐
         v                 v                 v
  ┌────────────┐   ┌────────────┐   ┌────────────┐
  │ gateway-   │   │ service-N  │   │  Center    │
  │ agent      │   │ + agent    │   │  :8080     │
  └─────┬──────┘   └─────┬──────┘   └─────┬──────┘
        │                │                │
        └────────────────┴────────────────┘
                         │
              ┌──────────┼──────────┐
              v          v          v
           Redis      Loki    Prometheus
```

启动：`./deploy/wsl/setup.sh start`  
验证：`./deploy/wsl/verify.sh`

Compose 文件：

- `deploy/docker/docker-compose.yml` — 完整栈
- `deploy/docker/docker-compose.wsl.yml` — WSL 资源优化覆盖

## 7. 本地开发快捷路径

```bash
# 混合模式：基础设施 Docker + 本地 Go
cd agent && go run ./cmd/agent -config configs/agent.yaml
cd center && go run ./cmd/center -config configs/center.yaml

# 单元测试
cd agent/pkg/retry && go test -v
```

## 8. 关键配置项

| 配置 | 位置 | 说明 |
|------|------|------|
| 响应时间阈值 T | `center/configs/center.yaml` | 高价值过滤 |
| 清单 Top-K | center strategy 配置 | 模式数量上限 |
| 日志发射间隔 | `agent/pkg/collector/collector.go` | 定向 2s / 全量 400ms |
| 缓存块大小 | agent yaml | 默认 64–256 MB |
| 退避参数 | agent yaml | baseDelay、maxRetries |
