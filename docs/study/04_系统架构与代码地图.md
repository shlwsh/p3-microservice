# 04 · 系统架构与代码地图

## 1. 仓库总览

```text
p3-microservice/
├── agent/          Sidecar 采集器
├── center/         日志中心
├── proto/          gRPC/Protobuf 定义
├── deploy/         Docker Compose、OpenResty、WSL 脚本
├── experiments/    实验配置、脚本、L0 结果
├── figures/        论文图表和绘图脚本
├── latex/          JOS/中文 LaTeX 源稿
├── docs/           L1 文档、版本化 PDF、study
├── scripts/        编译、实验、文献核验脚本
└── reviews/        多角色评审报告
```

## 2. Agent 模块

入口：`agent/cmd/agent/main.go`

| 包 | 职责 |
|----|------|
| `config` | YAML 配置加载 |
| `collector` | 应用日志采集、定向/全量发射间隔 |
| `matcher` | URL 模式和日志级别匹配，空清单 ERROR+ 兜底 |
| `cache` | 固定缓存块、环形队列 |
| `retry` | 指数退避 |
| `uploader` | gRPC 上传与 BoltDB 本地兜底 |
| `monitor` | CPU/内存压力感知 |
| `metrics` | Prometheus 指标 |

配置样例：

- `deploy/docker/gateway-agent-config.yaml`
- `deploy/docker/service-agent-config.yaml`

## 3. Center 模块

入口：`center/cmd/center/main.go`

| 包 | 职责 |
|----|------|
| `strategy/high_value_filter.go` | 高价值流量过滤 |
| `strategy/url_cluster.go` | URL 聚类和泛化 |
| `strategy/attention_list.go` | 关注清单生成 |
| `strategy/triple_transform.go` | 三次策略转换 |
| `redisstore/gateway_log_store.go` | Redis 网关日志读写 |
| `dispatch/rule_dispatch.go` | 规则广播 |
| `receiver/log_receiver.go` | 日志接收、去重、二次过滤 |
| `storage/loki.go` | Loki Push API |
| `grpcserver/server.go` | 上传与清单拉取 gRPC 服务 |

调试 API：

| 路径 | 说明 |
|------|------|
| `/api/v1/health` | Center 健康检查 |
| `/api/v1/attention-list` | 当前关注清单 |
| `/api/v1/agents` | Agent 列表 |
| `/api/v1/stats` | 实验统计接口 |
